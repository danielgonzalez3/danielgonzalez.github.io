
## Overview

The portfolio showcases Daniel’s education, work experience, technical skills, and projects. It also includes a contact form, downloadable resume, and social media links (GitHub and LinkedIn). The layout is responsive and optimized for both desktop and mobile viewing.

----------

## Features

-   **Responsive Layout**: Built with Bootstrap for a mobile-friendly experience.
-   **Downloadable Resume**: Users can download Daniel’s resume directly from the website.
-   **Project Highlights**: Key projects in each work experience are showcased through concise project cards.
-   **Contact Form**: An integrated Formspree-based contact form allows visitors to send messages directly to Daniel’s email.
-   **Social Media Links**: Icons link to GitHub and LinkedIn profiles.

## Folder Structure

```
├── css
│   ├── bootstrap.min.css
│   └── styles.css
├── libs
│   └── font-awesome
│       └── css
│           └── font-awesome.min.css
├── js
│   └── scripts.min.js
├── content
│   └── daniel_gonzalez_resume.pdf
└── index.html
```

-   **css/** – Contains the Bootstrap library and custom style sheets.
-   **libs/font-awesome/** – Font Awesome library for icons.
-   **js/** – JavaScript files for interactivity (jQuery and custom scripts).
-   **content/** – Any downloadable documents (e.g., Daniel’s resume).
-   **index.html** – Main HTML file containing the portfolio structure.
